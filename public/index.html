About Ogham

Ogham is an alphabet used to write the early Irish Gaelic language. It is believed to date back as far as the 1st century AD. Typically it was carved into stones for use as territorial and burial markers.

Ogham should be read from bottom to top. For example, the Irish word "fáilte" would look at follows if represented using Ogham:

ᚃᚐᚔᚂᚈᚓ

FÁILTE

Since Ogham is based on old Irish it contains only 20 of the typical Latin characters and the letters J, K, V, W, X, and Y are absent. Certain sources such as this one suggest phonetic replacements can be used in place of the missing letters.

This tool uses the following rules to transliterate the missing 6 letters.

Missing Character Replacemnt Character(s)
J G
K Q
V F
W UU
X Z
Y I
